Guide Lines:

Don't delete or change anything within < and > characters unless asked to do so.

Don't delete any main parts of the template, like icons and other things. Always comment them by adding <!-- at the start and --> at the end of the code for that section.

You should use a text editor specifically built for coding. You can use basic notepad type editor but don’t curse me afterwards. So better to use a nice code editor. There are some pretty awesome code editors which you can use for free like Brackets, Atom, Notepad++ and Sublime Text 2. I am using Brackets for this tutorial.

When I say open index.html or minilio.css that means to open that file in the code editor (usually by right clicking).

Change Colors:

The Template has three categories of colors:

Primary Color: #ff6347 (Tomato)
Secondary Color: #000000 (Black)
Background Color: #ffffff (White)

Add A Project Using Code Generator:

NOTE: To get rid of all demo projects when setting up the template. Just delete all project folders in the work folder and delete all code between <div class="portfolio"> and </div> tags. Remember don't delete the tags themselves, only the code between them.

1. In the work folder of the template, create a folder for your project.
